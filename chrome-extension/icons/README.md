# Extension Icons

Place your extension icons in this directory:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Create Icons

You can use any image editor to create simple icons, or use online tools:
- https://favicon.io/
- https://www.canva.com/

Recommended design:
- Simple icon showing two browser windows or tabs
- Use purple/blue gradient colors (#667eea to #764ba2)
- Include a small arrow or sync symbol

## Temporary Solution

For testing, you can use any PNG images of the correct sizes, or create simple colored squares.
